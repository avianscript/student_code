Using Font Awesome

You can read the full documentation for Font Awesome here.

Font Awesome provides a large collection of icons that are easy to add to your code and can be styled much the same way text can be. You can browse all the freely available Font Awesome icons here.

Below is an example of a Font Awesome icon. This is identified by the fa-hand-pointer Font Awesome class.

Adding this icon to your HTML is really easy...

The first step is including the Font Awesome stylesheet in your HTML file. To do this, add the following to the section of your HTML file: 

                            <link rel="stylesheet" href="https://use.fontawesome.com/releases/v6.1.0/css/all.css">

The above is for vanilla HTML/Javascript.

If you are using Vue, you need to do this a little differently.

The first step is to add Font Awesome to your Vue project. You can use this version of npm install to do that: 

                            $ npm install --save-dev @fortawesome/fontawesome-free

Then add these imports to your main.js 

    import '@fortawesome/fontawesome-free/css/all.css'
    import '@fortawesome/fontawesome-free/js/all.js'
    ...
    // this or something similar should already be there
    new Vue({  router,  store,  render: h => h(App),}).$mount('#app')

Once you have added Font Awesome, either via Vanilla Javascript or Vue, you can use Font Awesome icons in your HTML like this:

                            <i class="fas fa-hand-pointer"></i>
