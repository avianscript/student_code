rgba Explained

The main ways we discussed to specify a color in CSS were by:

  • name [i.e. pink]
  • hex value [i.e. #ffc0cb]
  • rgb value [i.e rgb(255, 192, 203)]

There are, however, some other options for how to specify color, including rgba.

rgba woks the same way as rgb but adds an additional parameter (referred to as the alpha value) for opacity.

Let's see how this works...

The example below uses these styles:

            
    .box-rgba {
        background-color: blueviolet;
        color: white;
        text-align: center;
        padding: 15px 5px;
        margin: 1px;
        font-size: 40px;
    }
    
    .box-rgb-inner {
        background-color: rgb(139, 0, 0);
        display: inline-block;
        position: relative;
        width: 50px;
        height: 50px;
        left: -50px;
    }

The HTML we are using looks like this:

 
    <div>
        <span class="box-rgba">Hello!</span>
        <span class="box-rgba-inner"></span>
    </div>

Here's what this looks like:

Hello!

The red box uses a relative position and specifies its left coordinate as -50px, which effectively puts it on top of the purple box.

Now let's create an additional style class and use that class for the inner box instead of box-rgb-inner.

    
    .box-rgba-inner {
        background-color: rgb(139, 0, 0, .50);
        display: inline-block;
        position: relative;
        width: 50px;
        height: 50px;
        left: -50px;
    }

Here's what we see now:

Hello!

Notice that you can now see through the red box to what's below it. In the style above we used rgba rather than rgb for the red box's color and specified an alpha value of .50, which means 50% opaque. Since it is only 50% opaque, we can see through it!

You can use rgba to accomplish this sort of thing. The alpha value is always a value between 0 and 1 which specifies the opacity of the color.
