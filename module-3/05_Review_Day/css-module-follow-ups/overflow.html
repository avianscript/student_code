Here is the base CSS used in these overflow examples: 
        .overflowdiv {
            width: 500px;
            height: 100px;
            border: 5px solid black;
            background-color: #6d9eeb;
            margin: 10px;
        }

As you can see, there is too much text for the width/height we have defined and the text continues outside of that area. Let's look at some options for handling this more gracefully.

Let's override the overflow property and set it to hidden.

        #hiddenDiv {
            overflow: hidden;
        }

As you can see this basically hides any text that would have overflowed.

        #scrollDiv {
            overflow: scroll;
        }
Now we can access the text that would have overflowed using the scrool bar.

But in this case, we maty have scroll bars in both directions, depending on the browser and operating system, and we only need them in the vertical direction. We can set the overflow property to auto to show scroll bars that will allow us to access the hidden content, but only if needed.

        #autoDiv {
            overflow: auto;
        }
Now scroll bars will be added only when content overflows - hortizontally, vertically, or in both directions.

Sometimes, we might not want our text to wrap. If we want to prevent word wrap, we can set the white-space property to nowrap. 
        #noWrapDiv {
            overflow: auto;
            white-space: nowrap;
            height: 40px;
        }
The style above prevents word wrap and also reduces the height of the element so it only holds one line. You can see that we have left overflow as auto to allow scrolling of the text that overflows.

Which FINALLY brings us to the answer to the original question about truncating text with an ellipsis. We can accomplish this by adding the text-overflow: ellipsis property. 
        #ellipsisDiv {
            overflow: hidden;
            white-space: nowrap;
            text-overflow: ellipsis;
        }
As you can see below, the text has been truncated with an ellipsis at the edge of the element. Notice though that we are still using white-space: nowrap in our style. Unfortunately, truncating in this way using just the properties we have discussed requires the text to not be wrapped. (It also requires the overflow propety to be set to hidden) It IS possible to accomplish the same thing with word wrapping but it is more complex and out of the scope of this discussion. If you'd like more info about this you can read about it here.
