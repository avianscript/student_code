Working with List Styles

There was a question about how to center bullet points along with the content of a iist. The answer to this was related to the CSS list-style property. Let's look at how this property works.

The list-style property is actually shorthand for three properties. Let's look at these properties one by one.

We will use this unordered list for our examples

    <ul>
        <li>
            Bread
        </li>
        <li>
            Milk
        </li>
        <li>
            Butter
        </li>
    </ul>

The list-style-type property allows to us to modify the list item marker. To use a square rather than a bullet point we can use the following style:

    .square-list {
        list-style-type: square;
    }

Now our list items are marked with squares:

Now let's look at the list-style-position property. Let's change our style to align our text to the right:

    .square-list-align-right {
        list-style-type: square;
        text-align: right;
    }

Here's what our list looks like now... uh-oh!

  • Bread
  • Milk
  • Butter

The reason for this is the list-style-position property.

The default for this property is outside. This means that the list item markers are outside of the main text of the list, so when we specify text-align: right only the text is aligned, leaving the markers left-aligned.

We can override list-style-position and set it to inside

    .square-list-inside-align-right {
        list-style-type: square;
        list-style-position: inside;
        text-align: right;
    }

Now our list item markers align with the text. The result below is still probably not exactly what we want but this illustrates the difference between list-style-position: outside and list-style-position: inside

We can use the list-style-image property to specify an image for our list item marker:

    
    .image-list {
        list-style-image: url("../images/eye-small.png");
    }

As mentioned before, the list-style property can be used as shorthand for all these properties. You can include multiple values in this property. Here's our right aligned style rewritten with only the list-style property:


    .square-list-inside-align-right-short {
        list-style: square inside;
        text-align: right;
    }
